## 数据结构的研究内容

通常，用计算机解题一个问题的步骤：

1. 具体问题抽象为数学模型（实质：a, 分析问题；b, 提取操作对象；c, 找出操作对象之间的关系；d, 用数学语言描述 -> 数据结构。）
2. 设计算法
3. 编程，调试，运行

首先，分析问题，提取操作对象。然后，找出操作对象之间的关系，用数学语言加以描述，建立相应数学方程。最后，求解数学方程：高斯消元法，差分法。。。

特点： 数据元素间的关系简单，计算复杂。
