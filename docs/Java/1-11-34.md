# Scanner的使用

```js
package ...;
import java.util.Scanner;

// Scanner: 扫描器
public class TestScanner {
 public static void main(String[] args) {
  // 1.创建一个扫描器
  Scanner scan = new Scanner(System.in);
  // 2.接收用户输入的字符串内容
  String s = scan.next(); // 方法就是一个功能
  System.out.println("您输入的内容为" + s);
 }
}
```
